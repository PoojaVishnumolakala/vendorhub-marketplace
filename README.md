# VendorHub Marketplace

An original MERN multi-vendor marketplace administration experience focused on orders, vendor performance, inventory and business metrics.

## Highlights

- Responsive marketplace operations dashboard
- Searchable order table and workflow status visualization
- Vendor-aware inventory and revenue KPIs
- Express/Mongoose order API with validated lifecycle states
- Secret-free demo mode and production-ready environment template

## Run

```bash
npm install
npm run install:all
npm run dev
```

Validate with `npm run build` and `npm test`.

## Stack

React, Vite, Node.js, Express, MongoDB, Mongoose, JWT-ready configuration and Vitest.
